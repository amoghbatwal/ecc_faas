If you click the "Submit" button, the form-data will be sent to my Cloud Function for respective type of input (text/graphical).

If the URL you enter is already present in the cache, the output will be fetched from there. If not (entering the URL for the first time), the output will be created from scratch and cached for future use.

The resulting output will display whether it was fetched from the cache or not.
